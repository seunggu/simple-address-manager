## 전화번호 관리 프로그램
LinkedList 를 이용한 간단한 전화번호 관리 프로그램

## 시작 화면
```cli
전화번호 관리
1.등록    2. 전체검색    3. 특정인검색    4.종료
```

## 등록

등록시 비밀번호를 검사하고 확인 된 후에 등록을 진행 할 수 있음

```cli
비밀번호: qwer1234

등록할 이름: ${이름} // 홍길동
전화번호: ${전화번호} // 010-1234-5678
```

## 비밀번호 실패
비밀번호는 3회까지 재시도 가능하며,
3회 실패시 프로그램이 종료됨

```
비밀번호(1회실패): ${비밀번호}
비밀번호(2회실패): ${비밀번호}
비밀번호(3회실패): 등록 할 수 없음! 프로그램을 종료합니다.
```

## 전체검색
```cli
<<전화번호목록>>
홍길동 010-1234-5678
문건영 010-4558-1234
나영빈 010-4124-2452
```

## 특정인검색
```cli
검색 할 이름: ${이름} // 홍길동

홍길동 010-1234-5678
```
